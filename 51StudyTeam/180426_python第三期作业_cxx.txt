# 6、请使用for循环编写一段程序，状态值有[0 ,1 ,2, 3]如果status =2，跳出本次循环，进行下一次循环；

 l=[0,1,2,3,4]
for i in l:
    if i==2:
        break
        print('终止循环')
    else:
        print(i)
        i=i+1

# 如果状态值为0，则终止循环。如果是其他值，则正常循环，并打印“运行正常”。
